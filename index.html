sw-image-upload





Proof-of-concept project for a native Web Component. This extends the input element with type="file".

It accepts only images and automatically shows a preview of the selected files (before they are uploaded to the server).

Double-clicking on an image preview displays it in a lightbox/modal.

Selecting an image and pressing Backspace or Delete will remove it. There is also a clear button.

The element uses an optional "maxuploads" attribute to prevent the user from uploading more than a given number of files.

Dynamically changing the maxuploads attribute (or maxUploads property) while the element is in use will also validate the number of files (and remove excess).

I tried to keep this as close to the native input as possible. Retrieving the element.files property returns a FileList.

Custom properties/methods for now:

  • files - Returns FileList
  • maxUploads - Modify the max (0 or null for unlimited)
  • clear() - Remove all images
  • removeFile(index) - Remove image by index

Once the script is loaded, the only code required to put this component on the page is:

<sw-image-upload></sw-image-upload>

document.createElement('sw-image-upload') will also work.
